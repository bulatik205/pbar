# PBar project

## Пару слов про проект:

> <i>PBar (скоращенно, от Progress Bar) - это анонимная страница для записи того, что хочет пользователь. Он записывает свои "Хочу" и это сохраняется в нашей базе. После, через время, можно выбрать то, чего добился пользователь. Просто так удалить успехи не получится. В конце некого промеждутка времени подразумевается отчет-статистика.</i>
